建造屬於自己的Hexo!

Posted on 2018-04-21

使用 Hexo 網誌框架 + GitHub Page，架設自己部落格！
經過不斷嘗試，終於成功建立(內牛滿面)，以下為超級菜鳥之Hexo不專業教學文

環境很重要

全部裝好裝滿，你才可以跑出一個部落格

你會需要安裝

  • Node.js
  • Git
  • Hexo
  • 註冊Github
  • Sublime Text ( 選配 )

先下載Node.js吧

  • 這是Node官網
  • 安裝好之後在”安裝的資料夾路徑”裡開啟cmd查看版本資訊(看是否安裝成功)

再來安裝Git

  • 下載Git_for_Windows-2.6.6-64-bit
  • 安裝到Adjusting your PATH environment這一步，如果不選use Git the windows Command Prompt，你就不能在windows cmd界面用Git，必須打開Git Bash

  • 安裝好之後在”安裝的資料夾路徑”裡開啟cmd查看版本資訊(看是否安裝成功)

  • 設定識別資料，在你安裝 Git 後首先應該做的事是設定使用者名稱及電子郵件。 這一點非常重要，因為每次 Git 的提交會使用這些資訊，而且提交後不能再被修改：
    可參考初次設定 Git

  • 來檢查看是否設定成功

去Github註冊

  • 這是Github官網
  • 然後開啟一個新的repository，並給予repository name
    你部落格的網址會是這樣https://github.com/你的帳號/你的repository_name

配置Hexo

  • 在電腦裡找一個舒適的地方….建立一個資料夾，阿我是在D槽建立D:/hexo
  • 進入資料夾裡按滑鼠右鍵打開cmd，就會進入該路徑底下
  • 輸入以下語法來直接安裝 Hexo
  • 安裝好查看版本資訊(看是否安裝成功)

選配Sublime Text

之後的檔案操作，你需要一個文字編輯器，要用記事本或是其他也是可以，我習慣用sublime

  • Sublime官網

開始Hexo!

上面環境都安裝成功就可以來玩玩hexo了

  • 也可以在資料夾裡按右鍵，選取Git Bash來操作
  • 輸入以下

你會看到生出一個ur_blog_name的資料夾，裡面好多資料XD

  • 再來路徑切到部落格資料夾，安裝環境依賴及需求

啟動伺服器(server)，看看內建hello-world

  • s (server)：啟動伺服器
  • -p (–port)：設定你要的port，8080、8090我常用，所以我改8050
  • 記得cmd路徑是要在你的部落格裏面，也可以Git Bash來操作

看到Hexo is running at http://localhost:8050/. Press Ctrl+C to stop. 出現
表示已啟動

打開網頁來看一眼部落格

url輸入 http://localhost:8050/ 來看看你的部落格

    例如: NexT

此時部落格資料夾裡的themes資料夾會多出一個next

  • 接著打開部落格的_config.yml檔案，我當然是用sublime打開
    把theme後面改成next，意思是告訴它我要使用next這主題
